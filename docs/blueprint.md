# **App Name**: KhushP

## Core Features:

- Photography Portfolio: Display photography works fetched dynamically from Google Drive via Cloudflare worker. Includes lightbox functionality to enlarge images.
- DJ Mixes Playlist: Stream DJ mixes fetched from Google Drive via Cloudflare worker. Includes audio player with play/pause and seek functionality.
- Projects Showcase: Display a collection of personal projects with links to live demos or external sites.
- About Me Section: Provide a detailed 'About Me' section showcasing skills, experience, and background.
- Contact Form: Allow users to send messages via a contact form, using Cloudflare Workers to handle email submissions without a traditional backend.
- Feedback Collection: Collect user feedback via a modal form, similarly leveraging Cloudflare Workers for submission.
- Code Style Summarizer: Allow users to paste code, then provide a plain-English summary. Uses generative AI tool to create summaries.

## Style Guidelines:

- Primary color: Strong, clean blue (#29ABE2) for trust and technology.
- Background color: Very light blue-gray (#F0F4F8), almost white.
- Accent color: Slightly darker, but analogous shade of turquoise (#2094B2) to maintain calm but draw attention.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and highly readable feel.
- Use a set of simple, outlined icons (e.g., from Tabler Icons) to represent different sections and links. Maintain consistent stroke width and style across all icons.
- Employ a clean, grid-based layout with generous spacing to create a professional and easily navigable site.
- Use subtle animations like fade-ins and smooth scrolling to enhance the user experience without being distracting.
